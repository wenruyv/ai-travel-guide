# 旅伴 - 整体系统架构设计（AI 攻略生成引擎版）

## 0. 产品定位与聚焦策略

> **一句话定位**：把"别人花 10 小时做的攻略"，变成"你 10 秒钟的事"。
>
> **核心心智**：输入"7 月带娃去日本 5 天预算 1 万"→ AI 一键生成结构化、可直接执行的旅行攻略（含每天交通/住宿/餐饮/景点时间）。

### 0.1 聚焦原则（5C 评审对齐）

| 评审维度 | 权重（海选）| 我们的策略 |
|---|---|---|
| 创新性 | 50% | **AI 攻略生成** + AI 行程助手 + AI 楼主分身，三层 AI 能力贯穿 |
| 聚焦性 | 30% | tabBar 保持 5 个但**首页重塑**：AI 攻略生成占核心入口，避免"大平台" |
| 价值贡献 | 20% | 攻略节点中嵌入**京东酒旅/装备推荐位**，生成即转化 |

### 0.2 核心交互闭环

```
AI 生成入口（首页 / 攻略 tab）
        ↓ 输入：目的地 + 天数 + 预算 + 同行人 + 偏好
        ↓
  AI 攻略生成引擎（LLM + 攻略模板库）
        ↓
  完整结构化攻略（时间轴 + 交通 + 住宿 + 餐饮）
        ↓
  攻略详情页（问真人楼主 / AI 行程助手 / 京东同款推荐）
        ↓
  旅账关联京东订单 / 找搭子 / 私聊楼主
```

## 2. 核心功能模块

### 2.1 🤖 AI 攻略生成引擎（核心入口）

> 区别于传统攻略 App 的最大创新点。

| 功能 | 说明 |
|------|------|
| 一键生成 | 输入目的地/天数/预算/同行人/偏好 → AI 生成完整攻略 |
| 模板库 | 内置 30+ 场景模板（亲子/情侣/独自/摄影/赏樱/滑雪…） |
| 节点级生成 | 每天的时间/交通/住宿/餐饮/景点分段生成 |
| 一键优化 | 生成后可对单一节点追加指令"换成民宿/加景点/避开人多的" |
| 京东种草位 | 节点中自动嵌入京东酒旅酒店 / 装备推荐位 |
| 复制并编辑 | 生成后用户可像普通攻略一样编辑、复制、分享 |
| 真实数据兜底 | 热门路线直接调用已验证的攻略数据（如 g1 川西/g3 关西） |

**AI 生成 Prompt 核心**：

```
[系统角色] 你是一位 10 年经验的旅行规划师，熟悉全球目的地。
[输入] 目的地={x}, 天数={n}, 预算={y}, 同行人={z}, 偏好={tags}
[约束]
- 每天 2-4 个节点，type ∈ {location, cost, transport}
- 必须包含 timeSlot / transport / accommodation / meal 四个字段
- 餐饮需具体到餐厅或菜品，不写"当地美食"这种泛词
- 交通需含方式+时长+参考费用
[输出] JSON 结构化攻略
```

## 4. API 设计原则

- RESTful 风格，JSON 数据格式
- JWT Token 认证（小程序登录后换取）
- 统一响应格式：`{code, message, data}`
- 分页：Page + Size 参数，返回 `{list, total, has_more}`
- 文件上传：独立上传接口，返回 URL 后关联业务数据
- **AI 接口特殊规范**：`POST /api/ai/guide/generate` 走 SSE 流式响应，支持中断重连

## 5. 技术选型理由

| 层级 | 技术 | 理由 |
|------|------|------|
| 前端 | 微信小程序 | 零安装门槛，社交传播强，原生能力全 |
| 后端 | FastAPI | Python 生态成熟，异步高性能，AI 集成方便 |
| 数据库 | PostgreSQL | 关系型数据为主，JSONB 支持灵活字段 |
| 缓存 | Redis | 会话、排行榜、消息队列、热点数据 |
| AI | Claude API | 推理能力强，适合复杂攻略生成和对话 |
| 向量 | pgvector / Redis | 攻略相似度匹配、语义检索 |
| 文件 | 腾讯云COS | 小程序生态原生支持，CDN 加速 |
| 部署 | 京东DevCloud / 腾讯云 | 根据赛事要求灵活选择 |

## 7. 京东商业化闭环（黑客马拉松核心亮点）

> 目标：将「旅伴」从社交工具升级为**京东酒旅业务的流量入口与转化载体**，用 AI 攻略生成降低用户决策成本，导流至京东酒店/机票/门票/度假及装备商品。

### 7.1 AI 驱动的五环闭环

```
🤖 AI 攻略生成 → 攻略详情种草 → 京东酒旅/装备转化 → 旅账关联订单 → 会员/任务促活
         ↑                                                          │
         └──────────────── 回流拉新/复购 ←──────────────────────────┘
```

| 环节 | 载体页面 | 商业价值 |
|------|----------|----------|
| ① AI 种草 | `pages/aiGuide/aiGuide` | 一键生成攻略，节点中嵌入京东推荐位 |
| ② 攻略种草 | `pages/guide/detail` | "订同款酒店" + "行前装备加购" |
| ③ 酒旅转化 | `pages/hotel/hotel` | 酒店/机票/门票/度假，PLUS 返京豆 |
| ④ 旅账关联 | `pages/bill/bill` | AI 自动分类 + 京东订单导入 |
| ⑤ 会员成长 | `pages/mine/mine` | PLUS 成长值、京豆余额、签到任务 |

### 7.2 关键玩法

- **AI 攻略种草位**：生成攻略时按 `tags`（高原/摄影/自驾/海岛/亲子…）映射 `gearByTag` 装备商品；`destination` 与酒店 `city` 交叉匹配"订同款"。
- **AI 行程助手加单**：用户问"我膝盖不好，稻城能换轻量路线吗" → AI 推荐 1-2 个备选酒店 → 一键跳京东酒旅。
- **搭子拼房立减**：结合搭子社交，酒店预订邀请搭子拼房再减 100 元，社交裂变驱动转化。
- **PLUS 返京豆**：订酒店最高返 8%，京豆可抵现，强化 PLUS 会员心智与复购。
- **旅账即订单**：京东酒旅订单一键进旅账，把"记账"变成"消费入口"。
- **任务/签到激励**：每日打开、浏览攻略、收藏酒店、完成预订均发京豆，直接提升 DAU。
